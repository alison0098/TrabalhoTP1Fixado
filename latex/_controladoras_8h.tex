\subsection{Detailed Description}
Arquivo que contém as classes do módulo que faz a comunicação entre a interface com o a lógica de negocio. 

O arquivo \hyperlink{_controladoras_8h}{Controladoras.\+h} contém os protótipos de todas as classes que permitem a comunicação entre a interface de usuário e a lógica de negócio.
